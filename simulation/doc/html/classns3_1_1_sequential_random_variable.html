Detailed Description

The Random Number Generator (RNG) that returns a sequential list of values.

Class SequentialRandomVariable defines a random number generator that returns a sequence of values. The sequence monotonically increases for a period, then wraps around to the low value and begins monotonically increasing again.

The following four parameters define the sequence. For example,

  • m_min = 0 (First value of the sequence)
  • m_max = 5 (One more than the last value of the sequence)
  • m_increment = 1 (Random variable increment between sequence values)
  • m_consecutive = 2 (Number of times each member of the sequence is repeated)
