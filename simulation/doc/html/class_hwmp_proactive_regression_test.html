Detailed Description

There are 5 stations set into a column, the center station is root.

Regression test indicates, that traffic goes from the first to the last stations without reactive route discovery procedure 

* Src                Root                 Dst
*           |         |<--------|-------->|         |          Proactive PREQ
*           |         |-------->|         |         |          PREP
*           |         |         |<--------|         |          PREP
*           |<--------|-------->|<--------|-------->|          Proactive PREQ
*           |-------->|         |         |<--------|          PREP
*           |         |-------->|         |         |          PREP
*           |         |         |<--------|         |          PREP
*  <--------|-------->|         |         |<--------|--------> Proactive PREQ
*  Note, that at this moment all routes are known, and no reactive
*  path discovery procedure shall be initiated
*  <--------|-------->|         |         |         |          ARP request
*           |.........|.........|.........|.........|
*           |         |         |<--------|-------->|          ARP request
*           |         |         |         |<--------|          ARP reply
*           |.........|.........|.........|.........|
*           |<--------|         |         |         |          ARP reply
*           |-------->|         |         |         |          DATA
*             ^ Further data is forwarded by proactive path
*
