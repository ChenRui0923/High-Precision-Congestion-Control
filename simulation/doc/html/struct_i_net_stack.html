virtual int INetStack::get_hz ( )
pure virtual

This is used so the simulator can call the stack timer_interrupt function the correct amount of times per second.

For example, lwip has a hz of 10, which it returns here to say that it's timer_interrupt should be called 10 times a second. FreeBSD uses 100, as does Linux 2.4, while Linux 2.6 uses 1000. (This is often configurable in the kernel in question, also.)

virtual const char* INetStack::get_name ( )
pure virtual

Should return a short one-word name of the stack.

Eg. Linux 2.4.x -> linux24, FreeBSD 5.x -> freebsd5. This can be used to identify output from a stack, for example a packet trace file.
