Detailed Description

Peering Management & HWM Protocol regression test Initiate scenario with 2 stations.

Procedure of opening peer link is the following: 

* <-----------|----------->   Broadcast frame
*             |----------->|  Unicast frame
*
*                                        !!! PMP routines:
* <-----------|----------->|             Beacon
*             |<-----------|             Peer Link Open frame 
*             |----------->|             Peer Link Open frame
*             |----------->|             Peer Link Confirm frame
*             |<-----------|             Peer Link Confirm frame
*             |............|             !!! Data started:
*             |<-----------|-----------> Arp Request
* <-----------|----------->|             Arp Request (fwd)
* <-----------|----------->|             PREQ
*             |<-----------|             PREP
*             |----------->|             ARP reply
*             |<-----------|             Data
*             |----------->|             Data
*             |............|             Some other beacons
*             |<-----------|             Data
*             |----------->|             Data
*             |............|             !!! Route expiration routines:
*             |<-----------|-----------> PREQ (route expired)
*             |----------->|             PREP
*             |<-----------|             Data
*             |----------->|             Data
*             |............|
*

At 10 seconds stations become unreachable, so UDP client tries to close peer link due to TX-fail, and UDP-srver tries to close link due to beacon loss
